# Java POO Exceptions – Metrics Analyzer

Proyecto educativo en Java que demuestra el uso de Programación Orientada a Objetos (POO) y manejo de excepciones personalizadas, simulando el cálculo de métricas en un contexto de backend.

## Características

- Organización por paquetes
- Excepciones personalizadas checked y unchecked
- Separación por capas (app, service, model, exception)
- Manejo de errores de entrada del usuario
- Código claro y estructurado

## Tecnologías

- Java
- Programación Orientada a Objetos
- Manejo de Excepciones

## Ejecución

1. Clonar el repositorio
2. Abrir en IntelliJ IDEA o Eclipse
3. Ejecutar la clase AppMetricas
